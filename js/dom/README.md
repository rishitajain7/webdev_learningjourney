# DOM Manipulation Project

This is a simple beginner-friendly project built using HTML, CSS, and JavaScript. The project focuses on learning how to interact with web pages using the DOM.

## Features
- Greets the user by displaying their name.
- Changes the background color of a box when it is clicked.
- Easy-to-understand JavaScript code for beginners.

## What I Learned
- Selecting elements using `getElementById()`
- Handling button and click events with `onclick`
- Reading user input using `.value`
- Updating text using `innerText`
- Changing styles dynamically with `style.backgroundColor`

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the project in your browser.
2. Enter your name in the input field.
3. Click the **Greet** button to display a personalized greeting.
4. Click any color box to change its background color.

This project was created to practice the basics of JavaScript DOM manipulation and event handling.
