
# Laundry Service Website

## Description

This is a beginner-friendly Laundry Service Website created using **HTML, CSS, and JavaScript**. The website allows users to explore laundry services, add services to a cart, book a service, and subscribe to a newsletter.

## Features

* Responsive Home Page
* Navigation Bar
* Laundry Services List
* Add Items to Cart
* Total Price Calculation
* Booking Form Validation
* Newsletter Subscription
* About Us Section
* Footer with Contact Information

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Pages

* **Home Page** – Displays the hero section and company information.
* **Service Page** – Shows available laundry services, allows users to add services to the cart, and submit a booking form.
* **About Us Page** – Displays service highlights and includes a newsletter subscription form.

## JavaScript Functionality

* Add selected services to the cart.
* Automatically calculate the total amount.
* Validate booking form inputs.
* Prevent booking without selecting any service.
* Display a booking confirmation message.
* Validate newsletter form and display a successful subscription message.

## How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser.
4. Navigate through the pages using the navigation bar.
