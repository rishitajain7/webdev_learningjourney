# 🚀 Responsive Landing Page

A simple and modern responsive landing page built using **HTML** and **Tailwind CSS**. This project was created to practice responsive web design and learn how Tailwind's utility classes can be used to build clean, mobile-friendly layouts without writing custom CSS.

## 📌 Features

- 📱 Fully Responsive Design
- 🧭 Sticky Navigation Bar
- 🍔 Mobile-Friendly Navigation
- 🎨 Gradient Hero Section
- 🖼️ Responsive Hero Image
- ✨ Call-to-Action Buttons
- 📦 Feature Cards with Hover Effects
- 📄 Responsive Footer
- ⚡ Built Using Tailwind CSS CDN
- 🚫 No Custom CSS Used

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS (CDN)

## 📂 Project Structure

```
Responsive-Landing-Page/

```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or build process is required since the project uses the Tailwind CSS CDN.

## 📱 Responsive Design

The landing page is designed using Tailwind's mobile-first approach and adapts to different screen sizes:

- 📱 Mobile

- 🖥️ Desktop
